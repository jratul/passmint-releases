# Passmint Releases

여권, TOPIK 성적표 등 각종 문서에서 정보를 추출하는 OCR 데스크톱 앱 **Passmint**의
배포용 저장소입니다. 빌드된 설치 파일(.exe)과 자동 업데이트용 파일만 이곳에
공개되며, 소스 코드는 비공개 저장소에서 관리됩니다.

## 다운로드 및 설치

1. [최신 릴리스](https://github.com/jratul/passmint-releases/releases/latest)
   페이지에서 `Passmint-Setup-x.x.x.exe` 파일을 다운로드합니다.
2. 다운로드한 설치 파일을 더블클릭해서 실행합니다. (별도 옵션 없이 자동으로
   설치되고 설치가 끝나면 앱이 바로 실행됩니다.)

### "Windows에서 PC를 보호했습니다" 경고가 뜨는 경우

Passmint는 별도의 코드 서명 인증서 없이 배포되어, 처음 실행 시 Windows
SmartScreen이 아래와 같은 경고를 표시할 수 있습니다.

> **Windows에서 PC를 보호했습니다**
> Microsoft Defender SmartScreen에서 알 수 없는 앱이 시작되지 않도록 차단했습니다.
> 이 앱을 실행하면 PC가 위험해질 수 있습니다.

이 경고는 정상이며, 아래와 같이 진행하면 됩니다.

1. 경고창의 **"추가 정보"** 를 클릭합니다.
2. 나타나는 **"실행"** 버튼을 클릭합니다.

한 번 실행을 허용하면 설치/실행이 정상적으로 진행됩니다.

## 실행

설치가 완료되면 바탕화면 또는 시작 메뉴의 **Passmint** 아이콘으로 실행합니다.

## 업데이트

### 자동 업데이트

- Passmint는 실행될 때마다 백그라운드에서 새 버전이 있는지 자동으로 확인합니다.
- 새 버전이 있으면 자동으로 다운로드되고, 다운로드가 끝나면 다음과 같은
  안내창이 표시됩니다.

  > 새 버전이 다운로드되었습니다. 지금 재시작하시겠습니까?

- **"지금 재시작"** 을 누르면 업데이트가 적용된 채로 앱이 다시 시작됩니다.
- **"나중에"** 를 누르면 다음에 앱을 다시 시작할 때 자동으로 업데이트가 적용됩니다.

### 수동으로 업데이트 확인하기

앱 상단 메뉴의 **도움말 > 업데이트 확인**을 클릭하면 즉시 최신 버전 여부를
확인할 수 있습니다.

- 이미 최신 버전이면 "현재 최신 버전을 사용하고 있습니다"라는 안내가 표시됩니다.
- 새 버전이 있으면 자동 업데이트와 동일하게 다운로드 후 재시작 안내가
  표시됩니다.

## 문제가 발생했을 때 (오류 로그)

앱 사용 중 오류가 발생하면 상단 메뉴의 **도움말 > 오류 로그 폴더 열기**로
로그가 저장된 폴더를 바로 열 수 있습니다. 문제를 보고할 때 이 폴더의 로그
파일을 함께 전달해주세요.
