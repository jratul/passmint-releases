# Passmint Releases

여권, TOPIK 성적표 등 각종 문서에서 정보를 추출하는 OCR 데스크톱 앱 **Passmint**의
배포용 저장소입니다. 빌드된 설치 파일(.exe)과 자동 업데이트용 파일만 이곳에
공개되며, 소스 코드는 비공개 저장소에서 관리됩니다.

## 다운로드 및 설치

1. [최신 릴리스](https://github.com/jratul/passmint-releases/releases/latest)
   페이지에서 `Passmint-Setup-x.x.x.exe` 파일을 다운로드합니다.
2. 다운로드한 설치 파일을 더블클릭해서 실행합니다. (별도 옵션 없이 자동으로
   설치되고 설치가 끝나면 앱이 바로 실행됩니다.)

### "Windows에서 PC를 보호했습니다" 경고가 뜨는 경우

Passmint는 별도의 코드 서명 인증서 없이 배포되어, 처음 실행 시 Windows
SmartScreen이 아래와 같은 경고를 표시할 수 있습니다.

> **Windows에서 PC를 보호했습니다**
> Microsoft Defender SmartScreen에서 알 수 없는 앱이 시작되지 않도록 차단했습니다.
> 이 앱을 실행하면 PC가 위험해질 수 있습니다.

이 경고는 정상이며, 아래와 같이 진행하면 됩니다.

1. 경고창의 **"추가 정보"** 를 클릭합니다.
2. 나타나는 **"실행"** 버튼을 클릭합니다.

한 번 실행을 허용하면 설치/실행이 정상적으로 진행됩니다.

## 실행

설치가 완료되면 바탕화면 또는 시작 메뉴의 **Passmint** 아이콘으로 실행합니다.

- 실행하면 작은 안내 화면이 잠깐 뜨면서 업데이트를 확인합니다(보통 3초
  이내). 업데이트가 없으면 곧바로 메인 화면이 뜨고, 업데이트가 있으면
  다운로드와 재시작 여부 결정까지 이 안내 화면 단계에서 끝낸 뒤 메인
  화면이 뜹니다 — 화면을 다 쓰는 도중에 재시작 다이얼로그가 끼어드는
  일이 없습니다.

## 사용 방법

### 기본 흐름

1. **폴더 선택** — 화면 왼쪽 상단의 "폴더 선택" 버튼을 클릭해 처리할 파일이
   들어 있는 폴더를 선택합니다.
   - 파일명에 `여권`이 포함된 파일은 여권으로, `한국어능력`이 포함된 파일은
     TOPIK 성적표로 자동 분류됩니다.
   - 이전에 같은 폴더를 처리한 적이 있다면 "이전 처리 결과 N건이 있습니다.
     불러올까요?" 확인창이 뜹니다. **불러오기**를 누르면 이미 처리된 결과를
     다시 OCR하지 않고 그대로 가져옵니다 — 앱을 껐다 켜거나 다른 작업을
     하다 돌아와도 처리 결과가 사라지지 않습니다.
     - 폴더 안 파일의 내용이 그대로일 때만 복원됩니다. 파일이 바뀌었거나,
       처리가 끝나지 않은 채로 앱이 종료된 파일은 복원되지 않고 다시
       처리됩니다.
2. **처리 시작** — "처리 시작" 버튼을 클릭하면 선택된 파일을 순서대로 OCR
   처리합니다. 처리 중 우측 상단의 **중단** 버튼으로 언제든 멈출 수 있습니다.
   - 처리 중에는 작업표시줄의 Passmint 아이콘에 크롬 다운로드처럼 진행률이
     초록색 막대로 표시됩니다.
   - 진행 표시줄 오른쪽에는 현재 처리 중인 단계가 실시간으로 표시됩니다.
     클릭하면 지금 처리 중인 파일의 누적 로그를 바로 확인할 수 있는 창이
     뜹니다 — 특정 파일이 오래 걸릴 때 어느 단계에서 멈춰있는지 확인할 수
     있습니다.
   - 처리 중에 창을 닫으려고 하면(X 버튼, Alt+F4, 메뉴의 종료 포함) "작업이
     진행 중입니다. 정말 종료하시겠습니까?" 확인 창이 떠서, 실수로 작업
     중인 내용을 잃지 않도록 합니다.
3. **결과 확인** — 처리가 끝나면 결과가 테이블로 표시됩니다. 행(row)은 학번,
   열(column)은 여권·TOPIK 각 필드로 구성됩니다.
4. **엑셀 내보내기** — "엑셀" 버튼으로 현재 테이블 데이터를 `.xlsx` 파일로
   저장합니다.

### OCR 결과 직접 수정

OCR이 잘못 인식한 값은 테이블에서 바로 수정할 수 있습니다.

- 수정하려는 셀을 **더블클릭**하면 편집 모드로 전환됩니다.
- 값을 입력한 뒤 **Enter** 또는 다른 셀 클릭으로 저장합니다.
- 수동으로 수정된 셀은 **노란색 배경**으로 강조 표시됩니다.
- 수정된 값은 엑셀 내보내기에도 그대로 반영됩니다.

> 셀 편집은 OCR이 성공한(초록색 "성공") 행뿐 아니라, 일부 필드만 인식에
> 실패해 "실패"로 표시된 행에서도 가능합니다(파일을 열 수 없는 등 파일
> 자체의 오류는 제외). 수정 후 남은 문제가 없으면 행이 자동으로 "성공"으로
> 바뀝니다.

### 실패한 파일 모아서 추출하기

처리 결과 중 "실패"나 "파일 오류"로 분류된 파일이 있으면, 결과 요약 줄의
**"실패 파일 추출"** 버튼으로 해당 파일들만 별도 폴더에 모아 복사할 수
있습니다.

- 버튼을 누르면 카테고리별(실패 / 파일 오류) 대상 폴더 이름을 입력하는
  창이 뜹니다. 기본값은 원본 폴더 이름 뒤에 `_실패`, `_파일오류`가 붙은
  이름으로 미리 채워지며, 필요하면 직접 수정할 수 있습니다.
- **"추출"** 을 누르면 원본 폴더와 같은 위치에 입력한 이름의 폴더가 생성되고
  해당 파일들이 복사됩니다.

### 파일 미리보기 및 열기

- 테이블의 파일명을 **클릭**하면 왼쪽 미리보기 패널에 이미지/PDF 첫 페이지가
  표시됩니다.
- 파일명을 **우클릭**하면 컨텍스트 메뉴가 나타납니다.
  - **연결 프로그램으로 열기** — 해당 파일을 Windows 기본 앱(사진 뷰어, PDF
    리더 등)으로 바로 엽니다.

### 파일명 검색

결과 테이블 위 검색창에 파일명(일부)을 입력하면 크롬 찾기(Ctrl+F)처럼 동작합니다.

- 일치하는 개수가 "1/3"처럼 표시됩니다.
- **Enter**(다음) / **Shift+Enter**(이전) 또는 위·아래 버튼으로 매치 사이를
  이동하면 해당 행으로 자동 스크롤됩니다.
- 검색어와 실제로 일치한 파일명 셀만 **보라색**으로 표시됩니다.
- 검색 중에는 상태 필터가 자동으로 "전체"로 전환되어, 필터 때문에 매치를
  못 찾는 일이 없습니다.

## 업데이트

### 자동 업데이트

- Passmint는 실행될 때마다 백그라운드에서 새 버전이 있는지 자동으로 확인합니다.
- 새 버전이 있으면 자동으로 다운로드되고, 다운로드가 끝나면 다음과 같은
  안내창이 표시됩니다.

  > 새 버전이 다운로드되었습니다. 지금 재시작하시겠습니까?

- **"지금 재시작"** 을 누르면 업데이트가 적용된 채로 앱이 다시 시작됩니다.
- **"나중에"** 를 누르면 다음에 앱을 다시 시작할 때 자동으로 업데이트가 적용됩니다.

### 수동으로 업데이트 확인하기

앱 상단 메뉴의 **도움말 > 업데이트 확인**을 클릭하면 즉시 최신 버전 여부를
확인할 수 있습니다.

- 업데이트 확인 중에는 화면이 어두워지며 로딩 표시가 나타납니다.
- 이미 최신 버전이면 "현재 최신 버전을 사용하고 있습니다"라는 안내가 표시됩니다.
- 새 버전이 있으면 자동 업데이트와 동일하게 다운로드 후 재시작 안내가
  표시됩니다.

## 문제가 발생했을 때 (오류 로그)

앱 사용 중 오류가 발생하면 상단 메뉴의 **도움말 > 오류 로그 폴더 열기**로
로그가 저장된 폴더를 바로 열 수 있습니다. 문제를 보고할 때 이 폴더의 로그
파일을 함께 전달해주세요.
